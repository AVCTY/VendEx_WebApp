Enhancement

Some features we offered in this website to give you an extraordinary experience

  • Home
  • Enhancement 1

Layout to accommodate smaller screen size

  • The purpose of this enhancement is provide better alternate presentation of our web pages for smaller screen devices such as smartphones and tablets.
  • This is applied by using media queries and setting the max screen width and changing its layout according to the screen size.
  • For example of codes written in CSS, @media screen and (max-width: 1080px){attribute: value;}
  • The application of this enhancement can be seen consistently throughout every web pages including this web page.
  • This enhancement can be clearly observed in a few web pages such as:
    • Responsive form
    • Responsive timeline
    • Navigation for smaller screens

Animation of profile cards

  • The purpose of this enhancement is to add animation and interactivity to the user using CSS only
  • This is made possible by creating 2 surfaces of the profile cards and transforming the back side by 180 degrees horizontally then using backface-visibility: hidden; to hide the back side of the card
  • After that, flip the card by 180 degrees horizontally when the user hover over the card.
  • This section of enhancement can be seen in the profile cards section in About Us page.

Usage of CSS attributes that are not covered in lecture

  1. Transition attribute
    • This is used in manipulating the duration of the effect applied in the web pages such as the flipping of profile cards and the changing of colours of elements when hovered over them.
    • This is achieved by implementing transition: transform 0.6s; and transition-duration: 0.5s; in the CSS.
    • The described enhancement can be observed in the profle cards section in About Us page. and the learn more buttons in the back side of the cards.
  2. Pseudo class and element in CSS
    • Multiple different pseudo classes and elements were used throughout the whole web pages to specify the styles applied to the elements in certain condition. For instance, ::after, ::before, :hover, :link and :focus
    • For example, when a link is hovered, the colour of the link will change and in the navigation bar, when a dropdown link is hovered then it will display the dropdown content out.
    • Some example of web pages that has this enhancement in them are coloured border when the input field is focused and insert a '>' in between items in the breadcrumb.

External widget from 3rd party source which is Google Maps

  • The purpose of this enhancement is to provide the user a deeper understanding on the location of our company.
  • This is applied in our web page by getting the html tag from Google Maps
  • This can be seen in the contact section of About Us web page.
